# Linux Security Basics

## User and Permission Management
- Used chmod to manage file permissions
- Used chown to control file ownership
- Reviewed permissions using ls -l

## Process and System Monitoring
- Used ps and top to monitor running processes
- Checked active users with who and w
- Reviewed system logs for suspicious activity

## Security Practices
- Principle of least privilege
- Regular review of user accounts
- Monitoring authentication logs for anomalies

## Relevance to Security Operations
Understanding Linux permissions and processes helps detect unauthorized access, privilege misuse, and system compromise.
